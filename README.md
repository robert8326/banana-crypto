# banana-crypto

Django: <br>
    1 pip install -r requirements <br>
    2 Зайдите в файл src <br>
    3 python manage.py migrate <br>
    4 python manage.py runserver

Telegram bot: <br>
    1 Зайдите в файл telegram_bot <br>
    2 python bot.py
    3 Бот будет доступен по ссылке: t.me/banana_crypt_bot
